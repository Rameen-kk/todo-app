# Glassmorphism To-Do List App ✨

A beautiful, modern To-Do List web app with glassmorphism UI effect. Built using HTML, CSS, and Vanilla JavaScript.

### 🚀 Live Demo
**[Click Here to Use the App](https://rameen-kk.github.io/todo-app/)**

### ✨ Features
- Add new tasks instantly
- Mark tasks as complete by clicking on them
- Delete tasks you don't need
- Beautiful glassmorphism design
- Fully responsive - works on mobile & desktop
- Saves tasks in your browser

### 💻 Tech Stack
- **HTML5** - Structure
- **CSS3** - Styling with Glassmorphism effect
- **JavaScript** - Functionality

### 🛠️ How to Run Locally
1. Clone this repo: `git clone https://github.com/Rameen-kk/todo-app.git`
2. Open `index.html` in your browser
3. Start adding tasks!
